# Figure Scripts (removed)

The figure-script export feature has been **removed** from the package. It was a deprecated path for
generating standalone publication scripts, built on a device-path "group" model that the
item/collection rework replaced.

Its intended replacement is the saved workflow and editable figure model described in
[plans/workspace-vision.md](plans/workspace-vision.md). If you need an old generated script, recover
it from version control.
