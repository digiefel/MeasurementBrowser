# DataBrowser

## Where to begin

- Start with [How DataBrowser works](pipeline.md) to understand the data flow.
- Follow [Your first project](getting-started.md) for the smallest complete project.
- Use the [Registration API](registration.md) when a source contains several items or needs
  processing and analysis.
- Read [Metadata and collections](metadata-and-collections.md) to control querying, labels, and the
  browser tree.
- Use the [Type API](type-api.md) when your domain already has concrete Julia types.
- Browse [Examples](examples.md) for complete projects of increasing complexity.
- Keep the [API reference](reference.md) nearby when writing project code.
